# Personal Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Vite. This portfolio showcases my projects, skills, and experience in an elegant and user-friendly interface.

## Features

- **Responsive Design**: Optimized for all device sizes
- **Modern Tech Stack**: Built with React, TypeScript, and Vite
- **Interactive UI**: Smooth transitions and hover effects
- **Project Showcase**: Dedicated section to highlight key projects
- **Contact Section**: Easy way for potential clients/employers to reach out
- **Clean & Elegant Design**: Professional aesthetic with attention to detail

## Tech Stack

- **Frontend**: React, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Inline SVG Icons

## Sections

- **Hero**: Eye-catching introduction
- **About**: Personal information and background
- **Skills**: Technical expertise showcase
- **Projects**: Featured projects with live links
- **Contact**: Contact information and social links

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit `http://localhost:3000`

## Deployment

This project can be easily deployed to platforms like Vercel, Netlify, or GitHub Pages.
